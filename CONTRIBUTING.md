# Como Contribuir 🤝

Agradecemos imensamente o seu interesse em ajudar a melhorar a Calculadora de Construção! 

Como o projeto **ainda não está 100% finalizado**, existe muito espaço para melhorias — desde a exatidão dos cálculos e fórmulas de engenharia, até a arquitetura do código e usabilidade da interface. **Por isso, qualquer contribuição (seja código, documentação ou relato de bugs) é extremamente bem-vinda!**

## Passo a passo de como contribuir:

1. Faça um **Fork** deste repositório.
2. Crie uma nova branch com a sua melhoria ou correção: 
   ```bash
   git checkout -b feature/minha-melhoria
   ```
3. Faça suas modificações no código. Lembre-se de manter o estilo limpo e descritivo.
4. Salve suas alterações criando um commit:
   ```bash
   git commit -m "feat: adiciona nova regra de cálculo para fundação"
   ```
5. Envie (push) os seus commits para o seu Fork:
   ```bash
   git push origin feature/minha-melhoria
   ```
6. Abra um **Pull Request (PR)** neste repositório explicando de forma simples o que você alterou e por que.

Iremos revisar as suas alterações o mais rápido possível e integrar ao projeto. Muito obrigado pelo apoio!
