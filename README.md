# Calculadora de Construção 🏗️

Uma calculadora web responsiva para estimativa de custos de obras e viabilidade de construção civil. O sistema realiza cálculos paramétricos (geometria, materiais, divisão por etapas) com base em coeficientes reais e métricas do mercado (como CUB/SINAPI).

⚠️ **Aviso de Desenvolvimento:** 
Este projeto **não está 100% finalizado**. Ainda há bastante o que melhorar, tanto no refinamento da matemática da engenharia quanto na interface e organização do código. É um trabalho em andamento!

## Tecnologias Utilizadas
- React 18 + Vite
- JavaScript moderno (ES6+)
- Recharts (para gráficos e visualização de dados)
